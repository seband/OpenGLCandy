# OpenGLCandy
This project is an OpenGL app that aims to provide an implementation of *Normal Mapping*, *Shadow Mapping* & *SSAO*, using LWJGL 3.

## Feature-list
- [ ] Basic project stucture
- [ ] Buffer generation
- [ ] OBJ-importer
- [ ] Input-handler
- [ ] FBO-handler
- [ ] Normal Mapping
- [ ] Shadow Mapping
- [ ] SSAO
